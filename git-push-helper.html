الأمر الأساسي

git push https://YOUR_GITHUB_TOKEN@github.com/mshari-11/delivery-form-server.git main

استبدل YOUR_GITHUB_TOKEN بالـ Personal Access Token الخاص بك

حفظ الـ Token بشكل دائم

git remote set-url origin https://YOUR_GITHUB_TOKEN@github.com/mshari-11/delivery-form-server.git

بعد تنفيذ هذا الأمر، يمكنك استخدام: git push origin main

📚 دليل مرئي

خطوات إنشاء Personal Access Token:

  1. افتح GitHub.com وسجل دخول
  2. اضغط على صورتك ← Settings
  3. في القائمة الجانبية: Developer settings
  4. Personal access tokens ← Tokens (classic)
  5. Generate new token (classic)
  6. املأ البيانات واختر "repo"
  7. Generate token
  8. انسخ الـ Token فوراً!

❓ أسئلة شائعة

ماذا لو نسيت الـ Token؟

لا يمكن استرجاع Token قديم. أنشئ token جديد بنفس الخطوات.

هل الـ Token آمن؟

نعم، طالما لا تشاركه مع أحد. تعامل معه كـ password.

لماذا يبدأ بـ ghp_؟
